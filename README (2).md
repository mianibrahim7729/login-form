## 🆘 Troubleshooting

**"Friend code not found"**
- Make sure your friend opened the app at least once
- Their data needs to exist in localStorage first

**"Leaderboard is empty"**
- You're the first user on this device!
- Add friends to see more people

**"Can't add friend"**
- Check you typed the code correctly (6 characters)
- Codes are case-sensitive
- Can't add yourself

**"Lost my friend code"**
- Go to Settings → Statistics
- Your code is always displayed there
- It never changes!
